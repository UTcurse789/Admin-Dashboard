Connect the GA4 Data API

Paste a Google service-account JSON key, confirm the GA4 property ID, then connect. The property must explicitly grant this service account access inside GA4 Admin. If you enable local storage, the credentials stay on this browser only.

Important

Service-account private keys are sensitive. This pattern is intentionally client-side because you asked for direct browser fetches, but it should be limited to private admin usage.
